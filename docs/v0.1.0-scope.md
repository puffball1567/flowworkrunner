# v0.1.0 Scope

FlowWorkRunner v0.1.0 is a complete first release for deterministic in-process
work execution.

## Included

- In-memory graph model
- Ready-batch planning
- Cycle detection
- Callback-based task executor registry
- wait-on failure propagation
- Explicit rejection of empty graphs and non-terminal executor results
- Run reports with `schemaVersion = 1`
- FlowCaptain-safe `run` outcome with `ok`, `report`, and `errors`
- Plugin manifests, event sinks, and task gates
- JSON report export
- Tests, example, benchmark, design notes, and IP notes

## Not Included

- Shell command execution
- Remote workers
- Thread pool scheduling
- Persistent storage
- Database adapters
- Adaptive control
- Dashboard rendering

## Completion Checklist

- [x] Public API exports match the documented v0.1.0 scope

- [x] Plugin extension points exist without embedding paid or external behavior
- [x] Tests cover validation, planning, execution, wait-on behavior, callback
